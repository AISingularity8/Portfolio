Deploy to GitHub

Step 1: GitHub Repository Setup

Follow these steps to deploy your portfolio to GitHub Pages:

  1. Create a new GitHub repository at github.com/new
  2. Name it: portfolio or fintech-portfolio
  3. Make it public
  4. Don't initialize with README (we have one)

Step 2: GitHub Credentials

Generate token here (Check: repo, workflow)

Step 3: Alternative - Manual Deployment

If you prefer not to use the automated deployment:

  1. Download all files from your portfolio folder
  2. Push them to your GitHub repository
  3. Go to Settings ? Pages
  4. Select "Deploy from a branch"
  5. Choose main branch
  6. Your site will be live at: https://yourusername.github.io/portfolio
