?? BACKUP YOUR ADMIN DATA:

   Your projects, skills, and education are stored in browser localStorage.
   
   Export Admin Data:
   1. Open admin.html in browser
   2. Press F12 (open Developer Tools)
   3. Click "Console" tab
   4. Paste:
      const backup = localStorage.getItem('adminPortfolioData');
      console.log(backup);
   5. Copy the output
   6. Save to file: admin-data-backup.json
   7. Keep this file safe!

   Restore Admin Data:
   1. Open admin.html
   2. Press F12
   3. Go to Console
   4. Paste:
      localStorage.setItem('adminPortfolioData', 'YOUR_BACKUP_DATA');
      location.reload();
   5. Replace 'YOUR_BACKUP_DATA' with your saved data
   6. Done!
